Secuencias de Escape
    \' = Mostrar una comillas simple dentro de una cadena en la salida de información
    \" = Mostrar doble comilla dentro de una cadena en la salida de información
    \\ = Barra invertida, diagonal inversa, es para mostrar una back slash
    \n = Esto es una salto de línea
    \r = Retorno
    \t = Tabulación
    \b = Retroceso
    \f = Salto de página

Operador de Aumento
    ++ = Incrementa de uno en uno
    Ejemplo: Let z = ++a; //PreIncremento: evalua de izquierda a derecha y a la variable a(que puede tener el número 1)le va a sumar un número más, entonces a z le asigna el valor de la variable pero sumandole un número más

    Ejemplo: z = a++; //PostIncremento: la variable z recibe el valor que tiene la variable a, por ejemplo la variable a vale 2, z va a valer 2 pero para el próximo uso qe hagamos de la variable a ya le va a aumentar un número, osea que "a" va a valer 3, pero z va a quedar valiedo 2. Primero guarda el valor en la variable z y luego le suma 1 

Operador de Decremento
    -- = Decrementa de uno en uno
    Ejemplo: Let z = --a; //PreDecremento
    Ejemplo: z = a--; //PostDecremento
